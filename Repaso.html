What is HTML? It is not a programming language since it has no control structure, loops or conditionals However, it is a markup or tagging language that allows the creation of a web page in which we can put content such as videos, photos, etc ... although it does not have to do with the visual of the web since for the style or design we must use CSS. To work in HTML we must have a web browser to be looking at what we do, and we will also use a text editor. --------------------------------------------------------------------------------------------------------------------------------------------------------- ¿Qué es HTML? No es un lenguaje de programacion ya que no tiene estructura de control, bucles ni condicionales sin embargo es un lenguaje de marcada o etiquetado que permite la creacion de una pagina web en el que podemos poner contenido como videos, fotos, etc... aunque tampoco tiene que ver con lo visual de la web ya que para el estilo o diseño debemos usar CSS. para trabajar en HTML debemos tener un navegador web para estar mirando lo que hacemos, y usaremos tambien algun editor de texto. Curicó (en mapuche: kurü ko, ‘aguas negras’)? es una ciudad y capital de la Provincia de Curicó, ubicada en la Región del Maule en la zona central de Chile. Las comunas que tiene Curicó son: Teno, Romeral, Rauco, Molina, Sagrada Familia y para la costa Vichuquén, Licantén, Hualañé. Curicó es una de las urbes más pobladas del valle longitudinal o ventral chileno.
